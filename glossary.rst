========
Glossary
========

.. glossary::

   long short-term memory

      A type of recurrent neural network designed to deal with the vanishing
      gradient problem.

      * `Wikipedia <https://en.wikipedia.org/wiki/Long_short-term_memory>`__

   machine learning

      The research and development of computer programs that learn from data
      and make useful predictions on unseen data.

      * `Wikipedia <https://en.wikipedia.org/wiki/Machine_learning>`__
      * `Machine Learning Glossary <https://developers.google.com/machine-learning/glossary/#machine-learning>`__
